sumix

What's sumix?

sumix is a Linux distribution based on Tiny Core Linux. The objective of this distribution is providing an easy way for trying softwares by SumiTomohiko.

Included Softwares

  • Flourish (action game)
  • Yog (script language)

Known Problem

Some PC show you dark desktop.

To solve this problem, give tinycore xsetup command when boot.
